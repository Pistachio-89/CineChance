# Phase 5 Context

## Задача
Дополнительный функционал фильтров Рекомендации.

## Требования

### 1. Переименовать "Мульт" в "Мульты"
- На странице Рекомендаций в FilterForm.tsx
- Везде, где используется этот тип контента

### 2. Добавить фильтры типов контента в Настройки пользователя
- На странице настроек профиля (profile/settings)
- Добавить секцию с переключателями для типов контента:
  - Фильмы (movie)
  - Сериалы (tv)
  - Аниме (anime)
  - Мульты (cartoon)
- Связать с фильтрами рекомендаций по аналогии со списками

## Аналогия со списками
В настройках пользователя уже есть связь списков (Хочу посмотреть, Смотрю, Просмотрено, Брошено). Нужно сделать аналогично для типов контента.

## Технические детали
- Типы контента: movie, tv, anime, cartoon
- Сохранение в базу данных (UserPreferencesSnapshot или подобное)
- Загрузка при инициализации фильтров рекомендаций
